# PRIMITIVE CATEGORY: gate

This directory contains the reference documentations for XMODEL's logic gate primitives, which model digital logic gates with xbit-type inputs and outputs.

## Contents
- and2_xbit           : A two-input AND gate with xbit-type input/output's.
- and_xbit            : A multi-input AND gate with xbit-type input/output's.
- buf_xbit            : A logic buffer gate with xbit-typed input/output's.
- delay_xbit          : An ideal delay element for xbit-typed signals.
- dff_rst_async_xbit  : A D flip-flop with xbit-type input/output's and asynchronous reset.
- dff_rst_sync_xbit   : A D flip-flop with xbit-type input/output's and synchronous reset.
- dff_set_async_xbit  : A D flip-flop with xbit-type input/output's and asynchronous set.
- dff_set_sync_xbit   : A D flip-flop with xbit-type input/output's and synchronous set.
- dff_xbit            : A D flip-flop with xbit-type input/output's.
- interp_var_xbit     : A delay-interpolating buffer between two xbit-typed signals with variable delay.
- interp_xbit         : A delay-interpolating buffer between two xbit-typed signals.
- inv_xbit            : A logic inverter gate with xbit-typed input/output's.
- latch_xbit          : A transparent latch with xbit-type input/output's.
- mux_xbit            : A multi-input multiplexer with xbit-type input/output's.
- nand2_xbit          : A two-input NAND gate with xbit-typed input/output's.
- nand_xbit           : A multi-input NAND gate with xbit-typed input/output's.
- nor2_xbit           : A two-input NOR gate with xbit-type input/output's.
- nor_xbit            : A multi-input NOR gate with xbit-type input/output's.
- or2_xbit            : A two-input OR gate with xbit-type input/output's.
- or_xbit             : A multi-input OR gate with xbit-type input/output's.
- tribuf_xbit         : A tri-state buffer with xbit-type input/output's.
- triinv_xbit         : A tri-state inverter with xbit-type input/output's.
- xnor2_xbit          : A two-input XNOR2 gate with xbit-type input/output's.
- xnor_xbit           : A multi-input XNOR gate with xbit-type input/output's.
- xor2_xbit           : A two-input XOR gate with xbit-type input/output's.
- xor_xbit            : A multi-input XOR gate with xbit-type input/output's.
