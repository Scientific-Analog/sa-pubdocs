# PRIMITIVE CATEGORY: vdt

This directory contains the reference documentations for XMODEL's variable-domain translator primitives, which convert between a clock and its properties such as frequency, phase, period, duty-cycle, and delay.

## Contents
- clk_to_delay        : A clock-to-delay domain translator.
- clk_to_duty         : A clock-to-duty domain translator.
- clk_to_freq         : A clock-to-frequency domain translator.
- clk_to_period       : A clock-to-period domain translator.
- clk_to_phase        : A clock-to-phase domain translator.
- delay_to_clk        : A delay-to-clock domain translator.
- duty_to_clk         : A duty-to-clock domain translator.
- freq_to_clk         : A frequency-to-clock domain translator.
- period_to_clk       : A period-to-clock domain translator.
- phase_to_clk        : A phase-to-clock domain translator.
