# PRIMITIVE CATEGORY: connect

This directory contains the reference documentations for XMODEL's connector primitives, which make connections between signals with different types.

## Contents
- bit_to_real         : A bit-to-real connector.
- bit_to_real_var     : A bit-to-real connector with variable logic-1 and logic-0 levels.
- bit_to_xbit         : A bit-to-xbit connector.
- bit_to_xreal        : A bit-to-xreal connector.
- bit_to_xreal_var    : A bit-to-xreal connector with variable logic-1 and logic-0 levels.
- real_to_bit         : A real-to-bit connector.
- real_to_bit_var     : A real-to-bit connector with variable logic-1 and logic-0 levels.
- real_to_temp        : Temperature assignment with a real-type value.
- real_to_xbit        : A real-to-xbit connector.
- real_to_xbit_var    : A real-to-xbit connector with variable logic-1 and logic-0 levels.
- real_to_xreal       : A real-to-xreal connector.
- temp_to_real        : Temperature measurement as a real-type value.
- temp_to_xreal       : Temperature measurement as an xreal-type value.
- xbit_to_bit         : An xbit-to-bit connector.
- xbit_to_real        : An xbit-to-real connector.
- xbit_to_real_var    : An xbit-to-real connector with variable logic-1 and logic-0 levels.
- xbit_to_xreal       : An xbit-to-xreal connector.
- xbit_to_xreal_var   : An xbit-to-xreal connector with variable logic-1 and logic-0 levels.
- xreal_to_bit        : An xreal-to-bit connector.
- xreal_to_bit_var    : An xreal-to-bit connector with variable logic-1 and logic-0 levels.
- xreal_to_real       : An xreal-to-real connector.
- xreal_to_temp       : Temperature assignment with an xreal-type value.
- xreal_to_xbit       : An xreal-to-xbit connector.
- xreal_to_xbit_var   : An xreal-to-xbit connector with variable logic-1 and logic-0 levels.
