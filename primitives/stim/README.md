# PRIMITIVE CATEGORY: stim

This directory contains the reference documentations for XMODEL's stimulus generator primitives, which generate analog or digital stimulus signals.

## Contents
- chirp_gen           : An analog chirp generator
- clk_gen             : A digital clock generator.
- const_xbit          : A constant-value generator with xbit-typed output.
- const_xreal         : A constant-value generator with xreal-typed outputs.
- dc_gen              : An analog DC generator
- exp_gen             : An analog exponential signal generator
- noise_gen           : A noise generator
- pat_gen             : A digital pattern generator.
- prbs_gen            : A digital PRBS generator.
- pulse_gen           : A digital pulse generator.
- pwl_gen             : An analog piecewise-linear stimulus generator.
- replay_xbit         : A reproducer of an xbit-type signal stored in a waveform file
- replay_xreal        : A reproducer of an xreal-type signal stored in a waveform file
- sin_gen             : An analog sinusoid generator
- step_gen            : An analog step generator
