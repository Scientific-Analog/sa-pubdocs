# PRIMITIVE CATEGORY: siph

This directory contains the reference documentations for XMODEL's silicon photonics primitives, which represent elements used in silicon photonic circuits and systems.

## Contents
- siph_cwlaser        : A continuous-wave laser source element.
- siph_cwlaser_var    : A tunable continuous-wave laser source element.
- siph_dircoupler     : An optical directional coupler.
- siph_phdetector     : A photodetector element.
- siph_phshifter      : An optical phase shifter element.
- siph_ringfilter     : An optical ring resonator filter.
- siph_ringfilter_var : An optical ring filter with modulation input.
- siph_ringres        : An optical ring resonator.
- siph_ringres_var    : An optical ring modulator.
- siph_term           : An optical termination element.
- siph_unit           : A unit port of optical waveguide.
- siph_waveguide      : An optical waveguide element.
- siph_ybranch        : An optical Y-shaped branch.
