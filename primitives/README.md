# Primitives

This directory contains the reference documentations for Scientific Analog's XMODEL primitives in Markdown format.

## Contents
- `/function`   : Function primitives
- `/circuit`    : Circuit primitives
- `/gate`       : Logic gate primitives
- `/stim`       : Stimulus generator primitives
- `/meas`       : Measurement probe primitives
- `/vdt`        : Variable Domain Translator (VDT) primitives
- `/connect`    : Connector primitives
- `/siph`       : Silicon photonics element primitives
