# PRIMITIVE CATEGORY: function

This directory contains the reference documentations for XMODEL's function primitives, which describe the functionality of analog/mixed-signal circuits.

## Contents
- abs_func            : An absolute value function for an xreal-type signal.
- adc                 : An ideal analog-to-digital converter.
- add                 : A multiple-input adder for xreal-typed signals.
- buffer              : A buffer for an xreal-typed signal.
- compare             : An analog clocked comparator.
- dac                 : An ideal digital-to-analog converter.
- delay               : A fixed transport delay for xreal-typed signals.
- delay_sel           : A selectable transport delay for xreal-typed signals.
- delay_var           : A variable transport delay for xreal-typed signals.
- deriv               : A derivative operator for xreal-typed signals.
- exp_func            : An exponential function for an xreal-typed signal.
- filter              : An analog linear filter for xreal-typed signals.
- filter_comb         : An analog comb filter for xreal-typed signals.
- filter_comb_var     : A variable comb filter for xreal-typed signals.
- filter_disc         : A discrete-time filter for xreal-typed signals.
- filter_disc_var     : A variable discrete-time filter for xreal-typed signals.
- filter_fir          : A finite impulse response filter.
- filter_sel          : A selectable analog linear filter for xreal-typed signals.
- filter_var          : A variable filter for xreal-typed signals.
- ilo                 : An injection-locked oscillator (ILO).
- integ               : An integration operator for xreal-typed signals.
- integ_mod           : An integration with modulo operator for xreal-typed signals.
- integ_rst           : An integration with reset operator for xreal-typed signals.
- inv_func            : A multiplicative inverse function (i.e. reciprocal function) for an xreal-typed signal.
- limit               : A limiting operator for xreal-typed signals.
- log_func            : A logarithmic function for an xreal-typed signal.
- multiply            : A multiple-input multipler for xreal-typed signals.
- poly_func           : A polynomial function for xreal-typed signals.
- poly_sel            : A selectable polynomial function for xreal-typed signals.
- power               : A power operator for xreal-typed signals.
- pwc_var             : A variable piecewise-constant (PWC) function for xreal-typed signals.
- pwl_func            : A piecewise-linear (PWL) function for xreal-typed signals.
- pwl_sel             : A selectable piecewise-linear (PWL) function for xreal-typed signals.
- pwl_var             : A variable piecewise-linear (PWL) function for xreal-typed signals.
- sample              : A sample-and-hold function for xreal-typed signals.
- scale               : A scaler for an xreal-typed signal.
- scale_var           : A variable scaler for an xreal-typed signal.
- select              : A signal selector (i.e. multiplexer) for xreal-typed signals.
- sin_func            : A sinusoidal function for an xreal-typed signal.
- slice               : An analog continuous-time comparator (slicer).
- sqrt_func           : A square-root function for an xreal-typed signal.
- transition          : An xbit-to-xreal transition filter.
