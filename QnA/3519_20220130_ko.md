# Q. 포아송 확률분포에서 랜덤 샘플 생성하는 방법

CMOS 이미지 센서를 모델링하면서, 포아송 확률분포에 따른 랜덤 노이즈를 생성해야할 필요가 생겼습니다. XMODEL에는 포아송 랜덤 노이즈를 생성할 수 있는 방법이 있나요?

## Answers

### A1

XMODEL에서는 다양한 확률분포로부터 샘플을 추출할 수 있는 DPI 함수들을 제공하고 있습니다.

```
void rand_seed(input int seed);
real rand_gaussian(input real mu, input real sigma);
real rand_uniform(input real a, input real b);
int rand_poisson(input real mu);
real rand_gamma(input real a, input real b);
int rand_binomial(input real p, input int n);
```

아래에 각 DPI 함수에 대한 설명을 드립니다.

* `rand_seed()`는 새로 생성되는 랜덤 샘플들의 seed 값을 설정합니다.
* `rand_gaussian()`는 평균 'mu'와 표준편차 'sigma'를 가진 가우시안 분포로부터 랜덤 샘플을 추출합니다.
* `rand_uniform()`는 'a'와 'b'사이의 균일 분포로부터 랜덤 샘플을 추출합니다.
* `rand_poisson()`는 평균 'mu'를 가진 포아송 분포로부터 랜덤 샘플을 추출합니다.
* `rand_gamma()`는 형태모수 'a'와 척도모수 'b'를 가진 감마 분포로부터 랜덤 샘플을 추출합니다.
* `rand_binomial()`는 확률 'p'와 시도횟수 'n'의 이항 분포로부터 랜덤 샘플을 추출합니다.

이중 `rand_poisson()`와 `rand_binomial()` 함수는 다른 함수들과 달리 각각 도착횟수와 성공횟수를 의미하는 정수의 값을 리턴함에 유의하기 바랍니다.

예를 들어, 아래의 SystemVerilog 코드는 신호 'clk'에 positive edge가 발생할 때마다 평균 10을 가진 포아송 분포에서 랜덤 샘플을 추출하여, 이 값을 변수 'sample'에 대입합니다.

```
int sample;
always @(posedge clk) begin
    sample = rand_poisson(10.0);
end
```
