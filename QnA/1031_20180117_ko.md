# Q. GLISTER 라이센스의 timeout 기능

GLISTER 라이센스가 일정시간 사용하지 않으면 자동으로 라이센스가 release되도록 하는 기능이 있는지요?

## Answers

### A1

예, 가능합니다. 자세한 내용은 GLISTER User Guide Section 2.1에 설명되어 있습니다.

요약해드리면, GLISTER 라이센스의 time-out 기간을 `$XMODEL_LICENSE_TIMEOUT` 환경 변수나 `xmodelLicenseTimeOut` SKILL 변수를 정의하여 분단위로 설정하실 수 있습니다. 기본값은 240분(4시간)입니다.

예를 들어, 아래의 명령을 Virtuoso 실행전에 Linux shell 상에서 실행하시거나,

```
setenv XMODEL_LICENSE_TIMEOUT 240
```

또는

```
export XMODEL_LICENSE_TIMEOUT=240
```

아래 라인이 `xmodel.cxt` 파일을 로딩한 이후에 실행되도록 `.cdsinit` 파일안에 포함시키시면 됩니다.

```
xmodelLicenseTimeOut = 240
```
