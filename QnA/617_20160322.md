# Q. Always raising new form windows on top

While using Virtuoso with XMODEL/GLISTER integration, I notice sometimes the newly-created dialog form windows hide behind the other existing windows, making it difficult to notice whether they have popped-up or not. Is there a way to fix this behavior?

## Answers

### A1

Yes. This is neither the Virtuoso’s problem nor XMODEL’s problem. It is the behavior defined by your windows manager.

The solutions vary depending on the window manager you are running on your operating system. For instance, if you are using Gnome, one solution is to execute the following command:

```
gconftool-2 --set /apps/metacity/general/new_windows_always_on_top --type bool true
```

Or, you can launch `gconf-editor` and use the GUI editor to do the same.
