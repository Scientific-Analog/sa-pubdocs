# Q. FSDB 파형 파일의 사이즈가 커지면 자동으로 파일을 분할하는 방법

신호들의 파형을 `$xmodel_dumpfile()` 명령을 통해 FSDB 형식으로 저장할때, 파일의 사이즈가 일정 크기 이상 커지면 새로운 파일로 전환되게 하는 기능이 있나요?

## Answers

### A1

네. XMODEL 2018.12버전부터 `$xmodel_dumpfile()` 명령이 `"sizelimit="` 옵션을 지원합니다. 이 `"sizelimit="` 옵션을 통해 생성되는 FSDB 파일의 크기를 메가바이트 단위로 지정할 수 있으며, 현재 파일의 크기가 이 크기에 도달하게 되면 자동으로 새 파일로 전환이 이루어집니다.

예를 들어, 아래의 경우,

```
$xmodel_dumpfile("xmodel.fsdb", "sizelimit=", 20);
```

시뮬레이션이 진행되면서, 아래와 같은 이름의 FSDB 파형 파일들이 현재 파일의 크기가 20MB에 도달할 때마다 순차적으로 생성됩니다.

```
xmodel_000.fsdb
xmodel_001.fsdb
xmodel_002.fsdb
...
```
