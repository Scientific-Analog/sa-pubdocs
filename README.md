# sa-pubdocs
Scientific Analog's Public Documentation Repository

This repository contains publicly shareable Scientific Analog's documentations.

## Purpose
These documents are published to improve machine readability and enable
LLM-based search, indexing, and retrieval.

## Contents
- `/brochures`  : product brochures
- `/manuals`    : product manuals
- `/papers`     : technical papers
- `/tutorials`  : tutorials
- `/appnotes`   : application notes
- `/QnA`        : Q&A docs
